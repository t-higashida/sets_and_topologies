\subsection{超限帰納法}
数学的帰納法は，自然数$n$に関するある命題$P(n)$が成立することを，
次の(１)，(２)を示すことによって証明する方法である．

\begin{description}
\item{(１)}　$P(1)$は真である．

\item{(２)}　任意の自然数$n>1$に対し，$n$よりも小さいすべての自然数$k$について$p(k)$が真であると仮定すれば，$p(n)$も真である．

\end{description}
この方法は，一般の順序数に対し，次のように拡張される．

\vspace{3mm}
\定理{６}
$\alpha$，$\beta$を$\alpha < \beta$なる序数とする．$\alpha \leqq \nu < \beta$なる順序数$\nu$に関する
ある命題$P(\nu)$に対し，次の(１)，(２)が証明されたと仮定する．
\begin{description}
\item{(１)}　$P(\alpha)$は真である．

\item{(２)}　$\alpha < \nu < \beta$を満たす任意の順序数$\nu$に対し，$\alpha \leqq \xi < \nu$
						なるすべての順序数$\xi$について$p(\xi)$が真であると仮定すれば，$p(\nu)$も真である．
\end{description}

このとき，$p(\nu)$は，$\alpha \leqq \nu < \beta$なるすべての$\nu$に対して成り立つ．

\証明
背理法によって証明しよう．$\alpha \leqq \nu < \beta$を満たすある$\nu$に対して$P(\nu)$が
成立しないものと仮定する．このとき，集合
\[
	M=\{ \nu \,| \, \alpha \leqq \nu < \beta , \nu \text{に対して$P(\nu)$が成立しない}\}
\]
は，整列集合$W\{\beta\}=\{\,\chi\,|\,\chi\text{ は順序数, }\,\chi	<	\beta	\,\}	$の空でない部分集合であるから，最初の元$\nu_0$が存在する．
$P(\nu_0)$は成立しないから，(１)より$\nu_0 \ne \alpha$，したがって$\alpha < \nu_0$である．
$\nu_0$は$M$の最初の元であるから，$\alpha \leqq \xi < \nu_0$なる任意の元$\xi$に対しては
$P(\xi)$が成立し，(２)により$P(\nu_0)$が成立する．これは矛盾である．
\■

定理６を用いて，順序数に関する命題を証明する方法を\textgt{超限帰納法}という．

定理６において，$\alpha=1$，$\beta=\omega$のときが，数学的帰納法である．
