\chapter{順序集合}

{\small この章では，順序とよばれる関係が与えられている集合，つまり順序集合について研究しよう．
集合の濃度に相当する概念として，順序集合には順序型というものが考えられる．同じ濃度を持つ集合であっても
，異なる順序型をもつことがあるので，順序型による分類は，濃度による分類よりも一層細かい分類である．

整列集合とよばれる順序集合は，自然数の順序集合を一般化して得られるものである．整列周後うの順序型は
順序数といわれ，大小や和や積が定義される．}

\section{順序集合}
本節では，実数の大小関係を一般化して得られる順序とよばれる関係について研究する．
まず，順序とは何かを説明し，順序を保つ全単射として，順序同型写像を定義する．
順序集合の間に定義される順序同型という関係は同値関係であり，この関係によって，
すべての順序集合を分割して得られる同値類を順序型と称する．

\subsection{順序集合}
２つの実数$a$，$b$をとるとき，$b$が$a$より大きい$(a < b)$か，大きくない$(a \nless b)$かのどちらかが
成り立つ．したがって$a < b$は，§1.4で述べた意味の“関係”である．この“関係”は，よく知られれた実数の
大小関係であり，さらに次の性質をもつ．

\begin{list}{}{}
		\item	1)　実数$a$，$b$，$c$に対し，
						\[
							a <b ,\, b <c \, \Rightarrow a < c
						\]
		\item	2)　任意の２つの実数$a$，$b$に対し，
						\[
							a<b, \, a=b, \, b<a
						\]
						のうち少なくとも１つが成り立ち，しかもどの２つも同時には成り立たない．
\end{list}
この大小関係を一般化して，次のように順序を定義する．

\定義{１}
集合$X$に，次の条件(1)，(2)を満たす関係$\prec$が定義されているとする．
\begin{list}{}{}
		\item	(1)　$X$の元$x$，$y$，$z$に対し
							\[
								x \prec y, \,\, y \prec z \Rightarrow x \prec z.
							\]
		\item	(2)　$X$の任意の２つの元$x$，$y$に対し，
							\[
								x \prec y, \,\, x=y, \,\, y \prec x
							\]
							のうちの少なくとも１つが成り立ち，しかもどの２つも同時には成り立たない．	
\end{list}

このとき，この関係$\prec$を\textgt{順序関係}（または単に\textgt{順序})といい，
この集合$X$を関係$\prec$に関する次\textgt{順序集合}という．
空集合も特別な場合として，順序集合と考える．

$\prec$が集合$X$の順序で，元$x$，$y \in X$の間に関係$x \prec y$が成り立つとき，
$y$は$\prec$に関して$x$より\textgt{後にある}，
$x$は$\prec$に関して$y$より\textgt{前にある}という．

集合$X$の上に順序$\prec$が与えられて得られる順序集合を
\[
	(X,\prec)
\]
と表わす．

　

\例{１}
実数全体の集合$\R$において，関係$x \prec y$が成り立つのは，大小関係$x \leqq y$
が成り立つとき，しかもそのときにがぎると定義すれば，これは１つの順序集合である．
この関係$\prec$を通常$\leqq$と表わす．

　

\例{２}
実数全体の集合$\R$において，関係$x \prec y$が成立するのは，大小関係$y < x$が成り立つとき，
しかもそのときに限ると定義すれば，これは１つの順序関係である．この関係$\prec$を$<^{-1}$と表わす．

　

\例{３}
空でない集合$X$の巾集合$P(X)$において，$X$の部分集合$A$，$B$に対し関係$A	\prec	B$が成り立つのは，
$A \subset B$が成り立つとき，しかもそのときに限ると定義すると，この関係は(1)を満足するが，
(2)を満たさないので，順序関係ではない．


	{\small 
\注意{1}
書物によっては，定義１で述べた“順序”のことを\textgt{全順序}または\textgt{線形順序}
ということがある．
}

　


\定義{２}
２つの順序集合$(X,\prec)$，$(X',\prec')$が\textgt{等しい}といわれるのは，次の２つの
条件が成り立つときである．
\begin{list}{}{}
		\item	(１)　２つの集合$X$，$X'$は等しい．すなわち，$X=X'$．
		\item	(２)　$X$の任意の２元$x$，$y$に対し，$x \prec y \Longleftrightarrow x \prec ' y$．
\end{list}
このとき，
\[
	(X,\prec)=(X',\prec')
\]
と書かれる．この否定は，
\[
	(X,\prec) \ne (X',\prec')
\]
と表わされる．

　

\例{４}
例１と例２における順序集合$(\R,<)$，$(\R,<^{-1}$は等しくない．なぜならば，実数の大小関係において，
$x < y \Longleftrightarrow y < x$は成立しないので，
定義２(２)が成り立たないからである．

　

\定義{３}
２つの順序集合$(X,\prec)$，$(A,\prec')$に対し

\begin{list}{}{}
		\item	(１)　$A$は$X$の部分集合，すなわち$A \subseteq X$
		\item	(２)　$A$の元$a$，$b$に対し，$a \prec' b \Rightarrow a \prec b$
\end{list}
の２つの条件が成り立つとき，$(A,\prec')$は$(X,\prec)$の\textgt{部分順序集合}(または単に
\textgt{部分集合})といわれ．
\[
	(A,\prec') \subseteq (X,\prec) \text{　　　（または} (X,\prec) \supseteq (A,\prec') \text{）}
\]
と記される．
$(A,\prec') \subseteq (X,\prec)$かつ$(A,\prec') \ne (X,\prec)$のとき，$(A,\prec')$は
$(X,\prec)$の\textgt{真部分順序集合}（または単に\textgt{真部分集合}）といわれ，
\[
	(A,\prec') \subset (X,\prec) \text{　　　（または} (X,\prec) \supset (A,\prec') \text{）}
\]
と表わされる．

　

\例{５}
有理数全体の集合$\Q$，整数全体の集合$\Z$，自然数全体の集合$\N$における順序を，いずれも例１のように実数の
大小関係$<$として定義すれば，次が成り立つ．
\[
	(\N,<) \subset (\Z,<) \subset (\Q,<) \subset (\R,<).
\]

有限順序集合や，$(\N,<)$，$(\Z,<)$などのような順序集合は，その順序もこめて，前にある元を左側に，後ろにある元を
右側にいっせいに並べて書き表すことができる．たとえば，次のように記す．
\begin{align*}
	&a_1 \prec a_2 \prec \cdots \prec a_n \text{　なるとき，　}(\{a_1,a_2,\cdots,a_n\},\prec)=(a_1,a_2,\cdots,a_n),\\
	&(\N,<)=(1,2,3,\cdots,n,\cdots)	\text{　（または単に　}(1,2,3,\cdots) \text{）},															 \\
	&(\Z,<)=(\cdots,-m,\cdots,-2,-1,0,1,2,3,\cdots,n,\cdots)
	\text{　（または単に　} (\cdots,-2,-1,0,1,2,3,\cdots) \text{）}.
\end{align*}
