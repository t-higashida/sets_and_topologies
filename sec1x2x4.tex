\subsection{集合の差　}
$A$，$B$を２つの集合とする．$A$に属し，$B$に属さない元全体の集合を$A-B$で表わし，
$A$と$B$との\textgt{差集合}（または単に\textgt{差}）という．すなわち，
\[
A-B=\{x \,|\, x \in A  \text{　かつ　} x \notin B \}.
\]

\例{１１}
$\Z-\N=\{0,-1,-2,\cdot\cdot\cdot,-n,\cdot\cdot\cdot\}=\text{（正でない整数全体の集合）}$．

\例{１２}
\begin{align*}
	A &=\{(x,y)	\,|\,	x,y \in \R,\,x^2+y^2<1 \} \\
	B &=\{(x,y)	\,|\,	x,y \in \R,\,(x-1)^2+y^2 \leqq 1 \}
\end{align*}
であるとき$A-B$は図の影の部分である．

\vspace*{4cm}
$ ]\,a,b\,[\,$，$[\,a,b\,]\,$，$\N$，$\Z$，$\Q$などが実数全体の集合$\R$の部分集合であるように，
集合はあるきまった集合の部分集合と考えられることが多い．$A$を集合$X$の部分集合とするとき，$X$と$A$との差
\[
X-A=\{x\,|\,x\in X \text{　かつ　}x \notin A \}
\]
を，$X$に関する$A$の\textgt{補集合}という．

\vspace*{5mm}
\textgt{例１３．}$\R$に関する$\Q$の補集合の元が無理数である．

\textgt{定理８．}$A$を$X$の部分集合とするとき，次が成り立つ．

（ⅰ）$A \cup (X-A) =X$

（ⅱ）$A \cap (X-A) =\varnothing$

（ⅲ）$X-(X-A)=A$

\textgt{証明}（ⅰ）
\begin{align*}
x \in A \cup (X-A) 	&\Longleftrightarrow x \in A \text{　または　} x \in X-A 							\\
					&\Longleftrightarrow x \in A \text{　または　} x \in X \text{　かつ　} x \notin A	\\
					&\Longleftrightarrow 	(x \in A \text{　または　} x \in X) \text{　かつ　} 
											(x \in A \text{　または　} x \notin A)						\\
					&\Longleftrightarrow x \in X \text{　かつ　} x \in X								\\
					&\Longleftrightarrow x \in X
\end{align*}
したがって$A \cup (X-A) \subseteq X$　かつ　$A \cup (X-A) \supseteq X$が成り立ち，（ⅰ）が示される．

（ⅱ）
\begin{align*}
x \in A \cap (X-A)	&\Longleftrightarrow x \in A \text{　かつ　}x \in X-A							\\
					&\Longleftrightarrow x \in A \text{　かつ　}(x \in X \text{　かつ　} x \notin A)\\
					&\Longleftrightarrow x \in X \text{　かつ　}(x \in A \text{　かつ　} x \notin A).
\end{align*}
ところが，$x \in A$　かつ　$x \notin A$ということはあり得ない．したがって，$A \cap ( X-A)=\varnothing$が示される．

（ⅲ）
\begin{align*}
x \in X-(X-A)	&\Longleftrightarrow x \in A \text{　かつ　} x \notin X-A								\\
				&\Longleftrightarrow x \in A \text{　かつ　} (x \notin X \text{　または　} x \in A)	\\
				&\Longleftrightarrow (x \in A \text{　かつ　} x \notin X) \text{　または　}
									 (x \in A \text{　かつ　} x \in A)								\\
				&\Longleftrightarrow x \in \varnothing \text{　または　} x \in A						\\
				&\Longleftrightarrow x \in A.
\end{align*}

\textgt{定理９．}$A$，$B$を$X$の部分集合とするとき，次の\textgt{ド・モルガン}(De Morgan)\textgt{の法則}が成り立つ．

（ⅰ）$X-A \cup B=(X-A) \cap (X-B)$

（ⅱ）$X-A \cap B=(X-A) \cup (X-B)$

\textgt{証明	}（ⅰ）$x \in X-A \cup  B \Longleftrightarrow x \in X \text{　かつ　} x \notin A \cup B$
\begin{align*}
\hspace*{8zw}&\Longleftrightarrow x \in X \text{　かつ　} (x \notin A \text{　かつ　} x \notin B)		\\
					&\Longleftrightarrow (x \in X \text{　かつ　} x \notin A) \text{　かつ　}
											 (x \in X \text{　かつ　} x \notin B)	\\
						&\Longleftrightarrow x \in X-A \text{　かつ　} x \in X-B 	\\
						&\Longleftrightarrow x \in (X-A) \cap (X-B)
\end{align*}
したがって，$X-A \cup B \subseteq (X-A) \cap (X-B)$ \text{　かつ　}$X-A \cup B \supseteq (X-A) \cap (X-B)$が成り立ち，
（ⅰ）が示される．

（ⅱ）$x \in X-A \cap B	\Longleftrightarrow x \in X \text{　かつ　} x \notin A \cap B							$
\begin{align*}
\hspace*{5zw}		&\Longleftrightarrow x \in X \text{　かつ　} (x \notin A \text{　または　} x \notin B)	\\
					&\Longleftrightarrow (x \in X \text{　かつ　} x \notin A) \text{　または　}
										 (x \in X \text{　かつ　} x \notin B)	\\
					&\Longleftrightarrow x \in X-A \text{　または　} x \in X-B 	\\
					&\Longleftrightarrow x \in (X-A) \cup (X-B)
\end{align*}

\textgt{定理１０．}$A$，$B$を$X$の部分集合とするとき，次の各条件は同値である．
\begin{description}
	\item{（１）}$A \subseteq B$
	\item{（２）}$X-B \subseteq X-A$
	\item{（３）}$A \cap (X-B)=\varnothing$
	\item{（４）}$(X-A) \cup B=X$
\end{description}

\textgt{証明}（１）$\Rightarrow$（２）の証：$A$の元はすべて$B$に属するという命題の対偶は，
$B$に属さない元はすべて$A$に属さない，すなわち，$X-B$に属する元はすべて$X-A$に属するという命題である．
したがって，（１）$\Rightarrow$（２）が得られる．

（２）$\Rightarrow$（１）の証：$X-B$の元はすべて$X-A$に属するという命題の対偶は，
$X-A$に属さない元はすべて$X-B$に属さない，
すなわち$X-(X-A)$に属する元はすべて$X-(X-B)$に蔵するという命題である．
定理８（ⅲ）を適用して，$A$に属する元はすべて$B$に属する．
したがて，（２）$\Rightarrow$（１）が得られる．

（１）$\Rightarrow$（３）の証：背理法によって証明する．$A \subseteq B$のとき，$ A \cap (X-B)$に属する元$x$が存在すると仮定すれば，$x \in A$　かつ　$x \notin B$である．これは$A \nsubseteq B$を意味するから，矛盾である．よって，$A \cap (X-B)=\varnothing$．

（３）$\Rightarrow$（１）の証：背理法によって証明する．$A \nsubseteq B$と仮定すると$B$に属さない$A$の元が存在する．このとき，$x \in A \cap (X-B)$で（３）に矛盾するから，$A \subseteq B$．

（３）$\Leftrightarrow$（４）の証：
\begin{align*}
A \cap(X-B)=\varnothing &\Leftrightarrow X-(A \cap(X-B))=X	&\because X-X=\varnothing	\text{，}
																																			X-\varnothing=X							\\
						&\Leftrightarrow X=(X-A) \cup (X-(X-B))					&\because \text{定理９（ⅱ）より}			\\
						&\Leftrightarrow X=(X-A)\cup B									&\because \text{定理８（ⅲ）より}
\end{align*}
以上で（１）$\Leftrightarrow$（２）$\Leftrightarrow$（３）$\Leftrightarrow$（４）が示された．\hspace*{27zw}■

\textgt{系　}$A$，$B$を$X$の部分集合とする．

（ⅰ）次の３条件は同値である．
\[
A \cap B=\varnothing	\text{　，　} A \subseteq X-B \text{　，　} B \subseteq X-A
\]

（ⅱ）次の３条件は同値である．
\[
A \cup B=X	\text{　，　} X-A \subseteq B \text{　，　} X-B \subseteq A
\]

\textgt{証明}（ⅰ）定理１０において，$X-B$を改めて$B$とおくと，（３）（１）（２）が同値であることから，定理８（ⅲ）を用いて求める結果が示される．

	（３）$A \cap (X-B)=\varnothing$ 	&&&（３）′$A \cap B=\varnothing$ 	\\
	（１）$A \subseteq B$			  	&&&（１）′$A \subseteq X-B$		\\
	（２）$X-B \subseteq X-A$ 			&&&（２）′$B \subseteq X-A$
\end{tabular}

（ⅱ）定理１０において，$X-A$を改めて$A$とおくと（４）（１）（２）が同値であることから，（ⅰ）と同様にして求める結果が得られる．

\begin{tabular}{llll}
	（４）$(X-A) \cup B=X$				&&&（４）″$A \cup B=X$		\\
	（１）$A \subseteq B$				&&&（１）″$X-A \subseteq B$\\
	（２）$X-B \subseteq X-A$			&&&（２）″$X-B \subseteq A$
\end{tabular}
