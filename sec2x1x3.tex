\subsection{集合の濃度}
前項の考察により，有限集合における元の個数の概念が，対等な集合を完全に特徴づけることがわかった．つまり，
\[\text{
“２つの有限集合が同じ個数の元をもつための必要十分条件は，$A$と$B$が対等であること”
}\]
である．必ずしも有限ではない集合に対し，この事実を一般化して，２つの集合$A$と$B$が\textgt{同じ濃度をもつ}
ことを，$A$と$B$が対等であることと定義しよう．
上の命題において“個数”を“濃度”に置き換えたものが，この定義である．
これで，“同じ濃度を持つ”ことの定義はできたが，それでは，“濃度”とは何であろうか．
一口でいえば，対等な集合全体につけられた“名称”のごときものである．このことをもう少し正確に述べよう．

定理１の(ⅰ)，(ⅱ)，(ⅲ)は§1.4で述べた同値律に相当し，定理１は，すべての集合の間に定められた“対等である”という
関係$\sim$が同値関係であることを示している．このとき§1.4定理１６のようにして，すべての集合の集まりは，同値な
ものを集めて得られる類，つまり同値類に分割される（次の注意２参照）．この同値類を\textgt{濃度}と称する．

集合$A$の同値類，すなわち$A$と対等であるような集合全体を$A$の\textgt{濃度}といい，
記号$|A|=\{X : \text{sets}\,|\, X \sim A\}$で表わす．
すべての集合には，濃度がただ1通りに定められ（すなわち任意の集合$A$について，$A \in |A|$），
２つの集合の濃度が等しいのは，２つの集合が対等であるとき，しかもそのときに限ること，すなわち
\[
|A|=|B| \Longleftrightarrow A \sim B
\]
が成り立つが定義から直ちにわかる．

有限集合の濃度，すなわち有限集合の元の個数を\textgt{有限濃度}という．
空集合の濃度は0，1個の元から成る集合の濃度は1，n個（$n \in \N$）の元から成る集合の濃度は$n$である．
したがって，上の記号によれば，
\[
|\varnothing|=0 \textgt{，　}
|\{ 1 \}|=|\{\varnothing\}|=1 \textgt{，　}
|\{1,2 \}|=|\{\varnothing,\{\varnothing\}\}|=2 \textgt{，　}
|\{1,2,\cdot\cdot\cdot,n \}|=n
\]
となる．

濃度は，ドイツ文字の小文字
\[
\mathfrak{a},
\mathfrak{b},
\mathfrak{c},\cdot\cdot\cdot
\]
によって表わされる．

\vspace{3mm}
{\small
\注意{２}
上で濃度を定義する際に，“すべての集合の集まり”を考えた．
§1.4
\textgt{定理１６．}

$x \in X$のすべての同値類$[x]$について，次が成り立つ．
\begin{list}{}{}
	\item(ⅰ)　すべての$x \in X$に対し$x \in [x]$
	\item(ⅱ)　$[x]=[y] \Longleftrightarrow xRy$
	\item(ⅲ)　$[x] \ne [y] \Longleftrightarrow [x] \cap [y]=\varnothing$
\end{list}
は
，同値関係が定義されている１つの“集合”を，同値なものに分割することができることを保証している．
“すべての集合の集まり”は“集合”ではなく“領域”である（§1.3 例１８と注意５）が，この定理１６は領域の場合にまで
拡張されて，同値関係が定められている１つの“領域”は同値なものの類に分割され得る．
}
