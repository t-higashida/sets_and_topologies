\subsection{集合の共通部分}

２つの集合$A$，$B$のどちらにも共通に含まれる元があるとき，$A$にも$B$にも属する元全体からなる集合を
$A$と$B$との\textgt{共通部分}といい，$A \cap B$で表わす．すなわち，
\[
	A \cap B =\{x | x \in A \text{　かつ　} x \in B \} \text{　．}
\]

\textgt{例３．}$A=\{1,2,3\}$，$B=\{2,3,4,6\}$ならば，
\[
	A \cap B=\{2,3\}\text{　．}
\]

\textgt{例４．}$A$および$B$がそれぞれ図のような円の内部として表わされるとき，$A \cap B$は影の部分である.

（A）$\Rightarrow$の証：$A$の任意の元$x$をとると，$A \subseteq B$ より$x \in B$．ゆえに，$x \in A \cap B$であるから，$A \subseteq A \cap B$が成り立つ．（@）より$A \cap B \subseteq A$であるから，$A \cap B=A$が得られる．
$\Leftarrow$の証：（@）より$A \cap B \subseteq B$　であるから，仮定および定理２（B）よち，$A \subseteq B$　が示される．
\■

\pagebreak
\textgt{定理６．}$A$，$B$，$C$を任意の集合とするとき，次が成り立つ．

（@）$(A \cap B) \cap C=A \cap (B \cap C)$　\textgt{（結合法則）}

（A）$A \cap B=B \cap A$　　　　　　　\textgt{（交換法則）}

\textgt{証明}
（@）
\begin{align*}
x \in (A \cap B) \cap C &\Leftrightarrow (x \in A \text{　かつ　} x \in B) \text{　かつ　} x \in C  \\
						&\Leftrightarrow  x \in A \text{　かつ　}(x \in B  \text{　かつ　} x \in C) \\
						&\Leftrightarrow  x \in A \text{　かつ　}(x \in B \cap C) 					\\
						&\Leftrightarrow  x \in A \cap (B \cap C)
\end{align*}
（A）
\begin{align*}
x \in A \cap B 	& \Leftrightarrow x \in A \text{　かつ　}x \in B \\
				& \Leftrightarrow x \in B \text{　かつ　}x \in A \\
				& \Leftrightarrow x \in B \cap A
\end{align*}
\■
\textgt{問７．}任意の集合$A$に対し$A \cap A=A$が成り立つことを示せ．

\textgt{証明}
\begin{align*}
x \in A \cap A \text{なる任意の元$x$に対して} 	&\Leftrightarrow x \in A \text{　かつ　} x \in A \\
												&\Leftrightarrow x \in A
\end{align*}
\■
\textgt{定理７.}
$A$，$B$，$C$を任意の集合とするとき，次の\textgt{分配法則}が成り立つ．

（@）$A \cup (B \cap C)=(A \cup B) \cap (A \cup C)$

（A）$A \cap (B \cup C)=(A \cap B) \cup (A \cap C)$

\textgt{証明}
（@）
\begin{align*}
	x \in A \cup(B \cap C) 	&\Leftrightarrow x \in A \text{　または　}  x \in B  \cap C 					\\
							&\Leftrightarrow x \in A \text{　または　} (x \in B  \text{　かつ　} x \in C) 	\\
							&\Leftrightarrow ( x \in A \text{　または　} x \in B)\text{　かつ　}
											 ( x \in A \text{　または　} x \in C)							\\
							&\Leftrightarrow ( x \in A \cup B)\text{　かつ　} ( x \in A \cup C)				\\
							&\Leftrightarrow x \in (A \cup B) \cap (A \cup C)
\end{align*}
したがって，$A \cup (B \cap C) \subseteq (A \cup B) \cap (A \cup C)$　かつ　
$A \cup (B \cap C) \supseteq (A \cup B) \cap (A \cup C)$　が成り立つ．

（A）
\begin{align*}
	x \in A \cap(B \cup C) 	&\Leftrightarrow x \in A \text{　かつ　}  x \in B  \cup C 						\\
							&\Leftrightarrow x \in A \text{　かつ　} (x \in B  \text{　または　} x \in C) 	\\
							&\Leftrightarrow ( x \in A \text{　かつ　} x \in B)\text{　または　}
											 ( x \in A \text{　かつ　} x \in C)								\\
							&\Leftrightarrow ( x \in A \cap B)\text{　または　} ( x \in A \cap C)			\\
							&\Leftrightarrow x \in (A \cap B) \cup (A \cap C)
\end{align*}
