\subsection{濃度の大小}
有限集合における元の個数の大小の拡張として，濃度の大小を定義することができる．
まず，集合$A$が集合$B$の部分集合であるときには，$|A|\leqq|B|$と定める．
一般には，$A$が$B$の部分集合と対等であるとき，$|A|\leqq|B|$と定めよう．
$A$が$B$の部分集合と対等であることは，$A$から$B$への単射が存在することと同じであるから，
濃度の大小を次のように定義することができる．

\定義{１}
２つの濃度$\mathfrak{a}$，$\mathfrak{b}$が与えられたとし，$\mathfrak{a}=|A|$，$\mathfrak{b}=|B|$
を満たす集合$A$，$B$をとる．$A$から$B$への単射が存在するならば，
\[
	\mathfrak{a}	\leqq	\mathfrak{b}	\text{　（または$\mathfrak{b}	\geqq	\mathfrak{a}$）}
\]	
と定め，$\mathfrak{a}$は$\mathfrak{b}$より\textgt{大きくない}，$\mathfrak{b}$は$\mathfrak{a}$より
\textgt{小さくない}，という．

　

ところで，この定義が意味をもつためには，$\mathfrak{a}	\leqq	\mathfrak{b}$という関係が
$\mathfrak{a}$と$\mathfrak{b}$
だけによってきまること，つまり$\mathfrak{a}=|A|$，$\mathfrak{b}=|B|$なる集合$A$，$B$の選び方
に無関係であることを調べる必要がある．いま，$A	\sim	A'$，$B	\sim	B'$で$A$から$B$への単射
$f$が存在するものとする．仮定によって存在する全単射$g:A	\rightarrow	A'$，$h:B	\rightarrow	B'$を
用いて，
\[
	f'=h	\circ	f	\circ	g^{-1}:A'	\rightarrow	B'
\]
をつくれば，$f'$も単射であることがわかる．したがって，$A$，$B$のかわりに$A'$，$B'$を用いても，同じ大小関係
が得られるから，上の定義は意味をもつ．

　

\定義{２}
２つの濃度$\mathfrak{a}$，$\mathfrak{b}$に対し，$\mathfrak{a}	\leqq \mathfrak{b}$
かつ$\mathfrak{a}	\ne	\mathfrak{b}$
なるとき，
\[
	\mathfrak{a}	<	\mathfrak{b}
\]
と定め，$\mathfrak{a}$は$\mathfrak{b}$よりも\text{小さい}，$\mathfrak{b}$は$\mathfrak{a}$よりも\text{大きい}
という．

　

{\small \注意{１}
正の偶数全体の集合$E$は，自然数全体の集合$\N$の真部分集合であるが，§2.1例２で見たように，$|E|=|\N|$である．
したがって，無限集合に対しては，一般に$A	\subset	B	\Rightarrow	|A|	<	|B|$は成立しない．
}

　
\例{２}$n$を任意の自然数とする．$\{1,2,\cdots,n\}	\subseteq	\N	$であるから，
$n=|\{1,2,\cdots,n\}|	\leqq |\N|	=	\Aleph_0$である．ところで，§2.1例３により$n	\ne \Aleph_0$であるから，
$n	<	\Aleph_0$が得られる．

\vspace{3mm}
\例{３}$\N	\subseteq	\R	$であるから，
$\Aleph_0=|\N|	\leqq	|\R|	=\Aleph$．定理５により$\Aleph_0	\ne	\Aleph$であるから，
$\Aleph_0 <	\Aleph$が示される．


\vspace{3mm}
有限濃度について，$0<1<2<\cdots<n<\cdots$はわかっているから，例２，例３を合わせて次が得られる：
\[
0<1<2<\cdots<n<\cdots<\Aleph_0<\Aleph.
\]

\vspace{3mm}
\定理{１１}
$\Aleph_0$は無限濃度のうちで最小である．

\証明
濃度の定義より，任意の無限集合$X$が可算集合を部分集合として包含するkとを示せば十分である．
$X$の任意の元を１つとって，それを$a_1$とする．
次に，$X-\{a_1\}$から任意に１つの元を選んで，それを$a_2$と名づける．この操作を順次繰り返す．
一般に，$a_1$から$a_{n-1}	\,	(n \geqq 2)$までがすでに選ばれたとする．
$X$は無限集合であるから，$X-\{a_1,a_2,\cdots,a_{n-1}\}	\ne \varnothing$であり，
そこで$X-\{a_1,a_2,\cdots,a_{n-1} \}$から，任意に１つの元をとって，それを$a_n$と名付ける．
このようにして，$X$から元$a_1,a_2,\cdots,a_n,\cdots$を取り出せば，$A=\{a_1,a_2,\cdots,a_n,\cdots\}$
は$X$の可算な部分集合となる．
\■


{	\small 
\注意{２}
この証明において，$a_1$は$X$から任意に取り出された元で，$a_2$は$X-\{a_1\}$から任意に取り出された
元，$\cdots$，$a_n$は$X-\{a_1,\cdots,a_{n-1}\}$から任意に取り出された元，$\cdots$であるから，
実際に集合$A$を作るには，任意の選択を無限回行う必要がある．
このような操作は，\textgt{ツェルメロ(Zermelo)の選択公理}とよばれる次の公理によって理論上許される．
}

　

\textgt{公理　}
集合族$\{A_\lambda|\lambda \in \varLambda,A_\lambda \ne \varnothing\}$（ただし，$\varLambda$は添え字の集合)が
与えられたとき，写像$f:\varLambda \rightarrow \bigcup_{\lambda \in \varLambda}A_\lambda$で，
$\lambda \in \varLambda$ならば，$f(\lambda) \in A_\lambda$となるようなものが少なくとも１つ存在する．

　

実際，定理１１の証明において，集合族$\{X-\{a_1,\cdots,a_{n-1}\}|n \in \N \}$に対し，選択公理を適用すれば，
各$n \in \N$に対して$f(n) \in X-\{a_1,\cdots,a_{n-1}\}$となる写像$f$が存在するから，$f(n)=a_n$とおけばよい．

{\small \注意{３}
$\aleph_0 < \mathfrak{a} < \aleph$なる濃度$\mathfrak{a}$が存在するかどうかという問題は，\textgt{連続体問題}
といわれる．この問題に関しても，近年興味ある結果が得られれている．さらに深く知りたい読者は，たとえば
赤摂也：集合論入門，培風館，p57，62，162などを参照されたい．
}

　

\定理{１２}
濃度の間に関係$\leqq$について，次が成り立つ．

\begin{list}{}{}
		\item	(ⅰ)　$\mathfrak{a} \leqq \mathfrak{b},\quad	\mathfrak{b} \leqq \mathfrak{a} \quad		
								\Longrightarrow \mathfrak{a}=\mathfrak{b}$
		\item	(ⅱ)　$\mathfrak{a} \leqq \mathfrak{b},\quad	\mathfrak{b} \leqq \mathfrak{c} \quad	
								\Longrightarrow \mathfrak{a} \leqq \mathfrak{c}$
\end{list}

（定理１２（ⅰ）も\textgt{ベルンシュタインの定理}といわれる．）

\証明
(ⅰ)　$\mathfrak{a}=|A|$，$\mathfrak{b}=|B|$なる集合$A$，$B$をとる．
$\mathfrak{a} \leqq \mathfrak{b}$より$A$から$B$への単射が存在し，
$\mathfrak{b} \leqq \mathfrak{a}$から$B$から$A$への単射が存在する．
このとき定理１０により$A$と$B$は対等，すなわち$|A|=|B|$が成り立つ．

(ⅱ)　$\mathfrak{a}=|A|$，$\mathfrak{b}=|B|$，$\mathfrak{c}=|C|$なる集合$A$，$B$，$C$をとる．
$\mathfrak{a} \leqq \mathfrak{b}$より単射$f:A \rightarrow B$が存在し，
$\mathfrak{b} \leqq \mathfrak{c}$より単射$g:B \rightarrow C$が存在する．
このとき，合成$g \circ f:A \rightarrow C$は単射であるから，$|A| \leqq |C|$が成り立つ．
\■

\問{９}
$\mathfrak{a} \leqq \mathfrak{b}	\,	,	\,	\mathfrak{b} < \mathfrak{c} \Rightarrow \mathfrak{a} < \mathfrak{c}$
を証明せよ．

\証明
$\mathfrak{b} < \mathfrak{c} \Longrightarrow \mathfrak{b} \leqq \mathfrak{c}$だから，
仮定より
$\mathfrak{a} \leqq \mathfrak{c}$．
もし，$\mathfrak{a} = \mathfrak{c}$ならば，
$\mathfrak{b} \leqq \mathfrak{c}$，$\mathfrak{c} \leqq \mathfrak{b}$
より$\mathfrak{b} = \mathfrak{c}$となり矛盾．
よって
$\mathfrak{a} < \mathfrak{c}$
